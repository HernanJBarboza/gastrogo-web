# Configuración inicial de Firebase para gastronomia-web

1. Ve a https://console.firebase.google.com/ y usa el mismo proyecto creado para la app móvil.
2. Agrega una app web en Firebase y copia la configuración.
3. Instala las dependencias:
   - npm install firebase @angular/fire
4. Agrega la configuración en environments/environment.ts y en el módulo principal de Angular.
5. Consulta la documentación oficial de AngularFire para detalles de integración.
